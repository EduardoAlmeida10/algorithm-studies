# Algorithm Studies

This repository was created to document my journey studying algorithms.

The goal is to solve **100 problems in 100 days**, practicing one problem per day, with a focus on classic algorithms, complexity analysis, and the development of algorithmic thinking.

## Objective

The main objective of this repository is to study algorithms in a practical way, going beyond just implementation.

For each problem, the idea is to understand:

- What the problem input is;
- What the expected output is;
- What a brute force solution would look like;
- What the complexity of the brute force solution is;
- Which technique or algorithm can be used to improve the solution;
- How to write the code;
- How to analyze the final complexity.
